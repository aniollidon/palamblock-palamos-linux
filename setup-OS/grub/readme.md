En aquesta carpeta hi ha:
+ grub -> /etc/default/grub
+ themes -> /usr/share/grub/themes


Instal·lació
1. Copiar `themes/whitesur/` a `/usr/share/grub/themes/whitesur/`
2. Copiar grub a `/etc/default/grub`
3. Executar `update-grub`


Tema creat a partir de https://github.com/vinceliuice/grub2-themes (whitesur)
